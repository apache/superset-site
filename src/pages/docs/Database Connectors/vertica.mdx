Vertica
-------------


[Vertica](http://www.vertica.com/) connection parameters:

* **User Name:** UserName
* **Password:** DBPassword
* **Database Host:**
    * For Localhost : localhost or 127.0.0.1  
    * For On Prem : IP address or Host name
    * For Cloud: IP Address or Host Name
* **Database Name:** Database Name
* **Port:** default 5433


```
vertica+vertica_python://<UserName>:<DBPassword>@<Database Host>/<Database Name>
```

Other parameters:

* Load Balancer - Backup Host
